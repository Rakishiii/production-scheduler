Optimization Setup

This module defines the model objective, declared parameters, workstation configuration, and operational constraints used by the Python-based optimization system for production scheduling and resource allocation.

Optimization Objective

Objective Function: Minimize Total Production Makespan (Cmax)

The system generates the shortest feasible production schedule while satisfying machine capacity, labor availability, and production sequence constraints.

Makespan is calculated from the start of first job to the completion of the final job.

Processing Time Parameters (Standard Time with 30% Allowance)

Derived from the Time and Motion Study (Appendix C).

Scheduling Rules Applied
  • Fixed production sequence
  • Cutting -> Edging -> Routing -> Assembly
  • No overlapping jobs per machine
  • Total assigned processing time per workstation
Scheduling Horizon
  • Working Days per Week: 6
  • Hours per Day: 8
  • Planned Downtime: 1 hour per day
  • Effective Machine Time: 7 hours/day
Optimization Engine: Constraint-Based Scheduling Solver
Model Type: Deterministic Job-Shop Scheduling
