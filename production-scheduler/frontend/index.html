3.3 Indices

Let: i = 1,2,3,...,N (customer orders/jobs)

Each order already includes: Cabinet type, Dimensions, Quantity, Custom features

m = 1,2,3,4 (machines/workstations)

Machine / Workstation

1 CNC Cutting (man-operated)

2 CNC Edging (man-operated)

3 CNC Routing (man-operated)

4 Assembly-Finishing-Packing (manual station)

3.4 Decision Variables and Parameters

Decision Variables: S_i = Start time of order i

C_i = Completion time of order i

C_max = Makespan (maximum completion time)

Parameters: p_{im} = Processing time of order i on machine m

H_m = Available working time of machine m

3.5 Objective Function

Minimize C_max

C_max ≥ C_i for all i ∈ {1,...,N}

The model minimizes the maximum completion time among all customer orders to achieve the shortest overall production schedule.

3.6 Model Constraints

3.6.1 Machine Capacity Constraints

∑_{i=1}^{N} p_{i1} ≤ H_1

∑_{i=1}^{N} p_{i2} ≤ H_2

∑_{i=1}^{N} p_{i3} ≤ H_3

∑_{i=1}^{N} p_{i4} ≤ H_4

3.6.2 Operation Sequence Constraints

CNC Cutting → Edging → Routing → Assembly-Packing

S_{i2} ≥ S_{i1} + p_{i1}

S_{i3} ≥ S_{i2} + p_{i2}

S_{i4} ≥ S_{i3} + p_{i3}

3.6.3 Completion Time Definition

C_i = S_{i4} + p_{i4}

3.6.4 Makespan Constraint

C_max ≥ C_i for all i ∈ {1,...,N}

3.6.5 Non-Negativity Constraints

S_i ≥ 0, C_i ≥ 0, p_{im} ≥ 0
